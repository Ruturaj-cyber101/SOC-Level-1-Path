# SOC Workbooks and Lookups

**Section:** SOC Team Internals  

**Description:**  
This room introduces the use of workbooks and lookup tables in a SOC
environment. These tools help analysts organize investigations,
enrich alerts with additional context, and streamline analysis
through structured data and saved queries.

**Key Concepts Learned:**
- Purpose of SOC workbooks
- Using lookup tables for enrichment
- Adding context to alerts
- Improving investigation efficiency
- Standardizing analysis processes
