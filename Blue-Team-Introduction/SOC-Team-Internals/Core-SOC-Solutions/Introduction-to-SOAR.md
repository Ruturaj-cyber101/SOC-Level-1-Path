# Introduction to SOAR (Security Orchestration, Automation and Response)

**Section:** Core SOC Solutions  

**Description:**  
This room introduces Security Orchestration, Automation, and Response (SOAR)
platforms used in modern SOC environments. It explains how SOAR tools
integrate with SIEM and other security solutions to automate repetitive
tasks, streamline incident response, and improve efficiency.

**Key Concepts Learned:**
- What is SOAR
- Integration with SIEM and EDR
- Automation and playbooks
- Incident response orchestration
- Reducing alert fatigue
