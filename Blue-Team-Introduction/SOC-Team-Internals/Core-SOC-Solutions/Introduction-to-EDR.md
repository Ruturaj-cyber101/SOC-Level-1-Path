# Introduction to EDR (Endpoint Detection & Response)

**Section:** Core SOC Solutions  

**Description:**  
This room introduces Endpoint Detection and Response (EDR) solutions,
which monitor endpoint devices such as laptops and servers for
suspicious activity. It explains how EDR tools help detect, investigate,
and respond to threats at the endpoint level.

**Key Concepts Learned:**
- What is EDR
- Endpoint visibility and monitoring
- Detecting malicious behavior
- Threat hunting capabilities
- Incident response at endpoint level
