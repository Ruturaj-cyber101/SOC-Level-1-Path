# Elastic Stack: The Basics

**Section:** Core SOC Solutions  

**Description:**  
This room introduces the Elastic Stack (ELK Stack) as a log management
and SIEM solution. It explains how Elasticsearch, Logstash, and Kibana
work together to collect, process, analyze, and visualize security logs.

**Key Concepts Learned:**
- Components of Elastic Stack (Elasticsearch, Logstash, Kibana)
- Log ingestion and parsing
- Searching and filtering logs
- Creating visualizations and dashboards
- Using Elastic for security monitoring
