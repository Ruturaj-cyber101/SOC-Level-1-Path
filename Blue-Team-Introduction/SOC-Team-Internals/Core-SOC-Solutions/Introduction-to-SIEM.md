# Introduction to SIEM (Security Information and Event Management)

**Section:** Core SOC Solutions  

**Description:**  
This room introduces Security Information and Event Management (SIEM)
systems used in SOC environments. SIEM solutions collect, aggregate,
and analyze logs from multiple sources to detect security incidents
and generate alerts.

**Key Concepts Learned:**
- What is SIEM
- Log collection and aggregation
- Event correlation
- Alert generation
- Centralized monitoring
- Importance of log retention
