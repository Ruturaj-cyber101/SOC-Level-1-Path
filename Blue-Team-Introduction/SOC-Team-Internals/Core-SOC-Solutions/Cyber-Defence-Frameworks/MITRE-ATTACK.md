# MITRE ATT&CK Framework

**Section:** Cyber Defence Frameworks  

**Description:**  
This room introduces the MITRE ATT&CK framework, a globally
recognized knowledge base of adversary tactics and techniques
based on real-world observations. It helps SOC analysts map
attacker behaviour and improve detection strategies.

**Key Concepts Learned:**
- What is MITRE ATT&CK
- Tactics vs Techniques
- Enterprise ATT&CK matrix
- Mapping alerts to ATT&CK techniques
- Using ATT&CK for threat hunting
- Improving detection coverage
