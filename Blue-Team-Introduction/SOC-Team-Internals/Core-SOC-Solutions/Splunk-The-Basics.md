# Splunk: The Basics

**Section:** Core SOC Solutions  

**Description:**  
This room introduces Splunk as a SIEM platform used in SOC
environments. It focuses on understanding the Splunk interface,
searching logs, and using basic SPL (Search Processing Language)
queries to investigate security events.

**Key Concepts Learned:**
- Overview of Splunk architecture
- Searching logs using SPL
- Indexes and sourcetypes
- Filtering and time ranges
- Creating basic reports and dashboards
