# IDS Fundamentals

## Overview
This room introduces Intrusion Detection Systems (IDS), which are used to monitor network traffic and detect malicious activities or policy violations.

## Key Concepts
- IDS monitors network or system activities for suspicious behavior.
- It generates alerts when potential threats are detected.
- IDS does not block traffic (unlike IPS).

## Types of IDS
- Network-based IDS (NIDS): Monitors network traffic
- Host-based IDS (HIDS): Monitors activity on individual systems

## Detection Methods
- Signature-based detection (known attack patterns)
- Anomaly-based detection (deviation from normal behavior)

## Common Alerts
- Port scans
- Brute-force attempts
- Malware communication
- Suspicious traffic patterns

## SOC Analyst Perspective
SOC analysts investigate IDS alerts to determine if they represent real threats or false positives. IDS plays a key role in alert generation and early threat detection in a SOC.

## Key Takeaways
- IDS is essential for threat detection.
- It helps identify malicious activity in real time.
- Analysts must investigate alerts carefully to avoid false positives.
