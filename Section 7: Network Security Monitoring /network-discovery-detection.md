# Network Discovery Detection

## Overview
This room focuses on detecting network discovery activities performed by attackers. Network discovery is often an early stage in an attack where adversaries scan and gather information about systems and services.

## Key Concepts
- Network discovery involves identifying devices, services, and open ports.
- Attackers use scanning techniques to map the network.
- Detecting these activities helps prevent further exploitation.

## Common Techniques
- Port scanning (e.g., using tools like Nmap)
- Network scanning to identify live hosts
- Service enumeration
- Ping sweeps
