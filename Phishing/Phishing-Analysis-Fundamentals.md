# Phishing Analysis Fundamentals

## Overview
This room focused on the foundational concepts of phishing investigations within a Security Operations Center (SOC) environment. The objective was to understand how phishing attacks are structured, identified, and analyzed using a systematic investigation approach.

Phishing remains one of the most common initial access vectors used by attackers, making email analysis a critical SOC skill.

---

## Key Learning Objectives

- Understand different types of phishing attacks (email phishing, spear phishing, whaling)
- Identify common phishing indicators
- Analyze suspicious email components
- Extract and validate Indicators of Compromise (IOCs)
- Apply structured investigation methodology

---

## Core Concepts Covered

### 1. Phishing Indicators
- Suspicious sender address
- Spoofed domains
- Urgent or threatening language
- Malicious attachments
- Embedded suspicious URLs

### 2. Email Header Analysis
- Reviewing sender and reply-to addresses
- Checking email routing path
- Identifying spoofing attempts
- Examining authentication failures

### 3. Malicious Links & Attachments
- Detecting shortened or obfuscated URLs
- Recognizing suspicious file types
- Identifying social engineering techniques

### 4. Indicators of Compromise (IOCs)
- Malicious domains
- Suspicious IP addresses
- File hashes
- Email addresses used in campaigns

---

## Investigation Workflow (SOC Perspective)

1. Review email content and sender details  
2. Analyze email headers  
3. Extract URLs and attachments  
4. Validate artifacts using threat intelligence sources  
5. Identify and document IOCs  
6. Escalate or respond based on findings  

---

## Practical Skills Developed

- Structured phishing investigation approach
- IOC identification and documentation
- Analytical thinking from a defender’s perspective
- Understanding how phishing fits into the attack lifecycle

---

## Why This Matters for SOC Roles

Phishing is a primary initial access technique used by attackers. Developing strong email analysis skills improves:

- Early threat detection
- Incident response effectiveness
- Risk reduction within organizations

This room strengthened foundational blue team investigation capabilities essential for a SOC Analyst role.
