# Phishing Analysis: Tools

## Overview
This room focused on the practical tools used by SOC analysts to investigate phishing emails effectively. The objective was to understand how to safely analyze suspicious artifacts and validate Indicators of Compromise (IOCs) using structured methodologies.

The lab emphasized tool-based investigation techniques commonly used in real-world SOC environments.

---

## Key Learning Objectives

- Use email header analysis tools
- Investigate suspicious URLs safely
- Analyze potentially malicious attachments
- Extract and validate Indicators of Compromise (IOCs)
- Leverage threat intelligence platforms during investigations

---

## Core Tools & Techniques Covered

### 1. Email Header Analysis Tools
- Tracing sender IP address
- Reviewing email routing path
- Identifying spoofing attempts
- Checking authentication failures (SPF, DKIM, DMARC)

### 2. URL Investigation Tools
- Expanding shortened URLs
- Checking domain reputation
- Identifying malicious infrastructure
- Reviewing WHOIS and domain registration details

### 3. File & Attachment Analysis
- Identifying suspicious file types
- Checking file hashes
- Understanding safe analysis practices
- Recognizing common malware delivery techniques

### 4. Threat Intelligence Platforms
- Validating IOCs
- Checking IP/domain reputation
- Correlating artifacts across multiple sources

---

## SOC Investigation Workflow with Tools

1. Extract artifacts from suspicious email  
2. Analyze email headers using header analyzer  
3. Investigate URLs using safe analysis tools  
4. Validate domains/IPs via threat intelligence sources  
5. Check file hashes (if attachment present)  
6. Document confirmed IOCs  
7. Escalate or remediate as required  

---

## Practical Skills Developed

- Safe investigation of malicious artifacts
- IOC validation and enrichment
- Structured use of threat intelligence sources
- Improved confidence in handling phishing alerts

---

## Why This Matters for SOC Roles

Effective phishing investigation relies not only on identifying suspicious content but also on validating technical evidence using professional tools.

Mastering investigation tools improves:
- Alert triage efficiency  
- Accuracy of threat classification  
- Speed of incident response  
- Overall SOC effectiveness
