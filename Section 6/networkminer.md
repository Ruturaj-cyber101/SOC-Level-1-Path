# NetworkMiner

## Overview
NetworkMiner is a Network Forensic Analysis Tool (NFAT) used to analyze network traffic and extract files, credentials, and other artifacts from captured packets. It is commonly used in incident response and forensic investigations.

## Key Concepts
- NetworkMiner analyzes packet capture (PCAP) files.
- It extracts useful artifacts such as files, images, and credentials.
- It helps analysts understand communication between hosts without manually inspecting every packet.

## Important Features
- Host identification
- File extraction from network traffic
- Credential extraction
- Session reconstruction
- OS fingerprinting

## SOC Analyst Perspective
SOC analysts use NetworkMiner during investigations to quickly extract information from captured traffic. It helps identify compromised systems, recover transferred files, and analyze attacker activity.

## Key Takeaways
- NetworkMiner simplifies network forensic analysis.
- It automatically extracts valuable data from PCAP files.
- It is useful for incident response and malware investigations.
