# Network Traffic Basics

## Overview
This room introduces the fundamentals of network traffic and how data travels across networks. Understanding network traffic is essential for SOC analysts to monitor communication and detect suspicious activities.

## Key Concepts
- Network traffic refers to the data moving across a network.
- Data travels in small units called packets.
- Each packet contains important information such as source IP, destination IP, protocol, and port number.

## Important Protocols
Common protocols seen in network traffic include:
- HTTP / HTTPS
- DNS
- TCP / UDP
- ICMP

## SOC Analyst Perspective
SOC analysts monitor network traffic to identify malicious behavior such as suspicious connections, malware communication, data exfiltration attempts, and unusual traffic patterns.

## Key Takeaways
- Network traffic analysis helps detect cyber attacks.
- Packets contain useful forensic information.
- Understanding protocols and ports is essential for security monitoring.
