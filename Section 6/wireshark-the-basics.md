# Wireshark: The Basics

## Overview
This room introduces Wireshark, one of the most widely used network protocol analyzers. It allows security analysts to capture and inspect network packets to understand what is happening inside a network.

## Key Concepts
- Wireshark is a packet analysis tool used to capture and analyze network traffic.
- It helps visualize packet-level data travelling across a network.
- Analysts can inspect protocols, IP addresses, ports, and packet contents.

## Important Features
- Packet capture and inspection
- Protocol analysis
- Packet filtering
- Traffic statistics and visualization

## Common Protocols Observed
Some protocols frequently analyzed in Wireshark include:
- HTTP / HTTPS
- DNS
- TCP
- UDP
- ICMP

## SOC Analyst Perspective
SOC analysts use Wireshark to investigate suspicious traffic
